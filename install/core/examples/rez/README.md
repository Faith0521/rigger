CgRigtools pro rez package example
================================


Rez : https://github.com/nerdvegas/rez 

How to use
==========
1. install rez as described on github
2. Install cgrigtoolspro either through the main distributed executable
or by unzipping the supplied zip file to a dev location
 
3. Copy the package.py and rezbuild.py to the cgrigtoolspro root folder
4. cd cgrigtoolspro
5. rez build --install
